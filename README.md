# 🌿 Rentora — Smart Rental Exploration Platform

**Rentora** is a modern, human-centered rental exploration platform that helps users discover homes with transparency, real user reviews, and location-based insights.
Designed with a warm, earthy UI, Rentora focuses on trust, clarity, and better decision-making for renters.

## 🎯 Problem Statement

Renters often struggle to identify hidden issues such as poor maintenance, safety concerns, unreliable landlords, or misleading property listings.
The lack of a transparent, community-driven platform leads to poor decisions and negative living experiences.

---

## 💡 Solution

Rentora provides a platform where users can:

* Explore rental properties visually
* Read real user reviews
* Analyze safety and quality insights
* Make informed housing decisions

---

## ✨ Features

### 🏠 Property Exploration

* Browse rental listings with images, price, and location
* Advanced search and filtering
* Save/favorite properties

### 📍 Map-Based Discovery

* Explore properties geographically
* Understand neighborhood context

### ⭐ Review & Rating System

* Add reviews and ratings
* Tag issues (noise, safety, maintenance, etc.)
* View rating breakdowns

### 📄 Property Insights

* Amenities and property details
* Safety and quality scores
* Community-driven feedback

### 📊 Dashboard & Insights

* View personal activity
* Ratings analytics
* Review history

### 📩 Contact Owner

* Send inquiries directly
* View property summary while contacting
